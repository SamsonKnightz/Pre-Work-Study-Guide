HTML

  • HTML is a markup language
  • 1989-1990 HTML was created to help describe the structure of research papers
  • HTML uses tags to describe the structures of a webpage
  • P(paragraph) tags are used for grouping
  • The STYLE element defines the CSS style associated with a website
  • The BASE element defines the base URL for a webpage
  • The HEAD element contains information about the webpage (outline of a document) Main, H1, H2 etc. default size
  • ANCHOR tag is used to tag a links
  • The META element represents metadata(data about data) like information of website like refresh rate, author, keywords, description, viewport
  • The BODY element represents the visible content shown to the user
  • The SCRIPT element embeds client-side scripts in the HTML document

CSS

  • CSS stands for Casscading Style Sheets
  • CSS is a style sheet language
  • Used for glamour/cosmetics
  • CSS is not a markup language or a programming language
  • CSS selectively describes the content in the document
  • CSS's whole structure is known as the ruleset
  • Language used to describe the visual representation of documents by using rulesets
      • 4 main ruleset are
    1. SELECTOR is the chosen HTML element name to be styled
    2. DECLARATION is a single rule that specifies which element you want to style
    3. PROPERTIES is the style effect variable
    4. PROPERTY VALUE is presented after the colon which chooses one out of many possible appearances for a given property
  • Besides the selector, each ruleset must be wrapped in curly braces{}
  • A COLON is used to separate the property from its value or values
  • A SEMI-COLON is used to separate each declaration from the next one
  • You could also select multiple SELECTOR elements by using commas
      • Many other SELECTOR elements, just to list a couple
    1. The CLASS selector selects Multiple instances of the same class can appear on a page
    2. The ID selector selects an element based on the value of the element's id attribute
  • A PADDING indicates how much space we want around the content inside an element
  • A MARGIN indicates how much space we want around the outside of and element.
  • A BORDER is a solid line that is outside the padding
      • Some common DECLARATIONS are
    1. WIDTH
    2. background-color
    3. text-shadow
    4. display
  • To change background color, select html, BACKGROUND-COLOR as property, color as property value

Git

  • Used to connect and contribute information about coding
  • Stores projects created
  • Start with CD to change directory(where you want to go)
  • Git STATUS will show current status of your repository(will show differences or changes in repository)
  • GIT CHECKOUT is used to switch branches
  • Codes can be reviewed and edited and documented of changes
  • In order to make a PR, perform command in sequence of (git add -A), then (git commit -m "what changes was done"), then (pull origin main), then (git push origin (whatever the branch you are trying to commit))
  • LS is used to list what is currently in that location

Javascript

  • Invented by Brendan Eich
      • 4 conditional statements
    1. General Truth
    2. Future events
    3. Hypotheitcal situation
    4. Hypotherical outcome
  • An array is a single variable that is used to hold a group of data
  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive
  • A variable is a named container that allows us to store data in out code
  • Control flow is the order in which a computer executes code in a script
  • Each item in an array has a set value starting from 0
  • An (for) loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over
  • A FUNCTION is simply a set of instructions that tells the computer how to perform a certain task
  • The CODEBLOCK is the instructions for what the function should do. AKA FUNCTION BODY
  • To CALL the function by write function name followed by the instructions in parentheses
